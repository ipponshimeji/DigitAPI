Digit API Reference

Request URL

http://digitapiweb.azurewebsites.net/digit/v1.0/recognize

httpsはサポートしていません。動かしているWeb AppsプランだとSSL運用できない。

Request headers

Content-Typeapplication/json

認証は不要なので、認証系のヘッダは不要です。

Request body

application/json 
{ "url": "http://example.com/picture.jpg" }

判定したい画像リソースのURLをしていします。 サイズは4M以下にしてください。 POSTで送ってください。

Emotion APIみたいにapplication/octed-streamを直接送りつけられるようにした方がいいかな。

Response 200

成功すると、以下のようなJSONがapplication/jsonで帰ってきます。 見たまんまです。
